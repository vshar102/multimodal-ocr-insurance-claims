# Contributing

This is a personal portfolio / learning project. It is not currently seeking
external contributions, but issues and suggestions are welcome.

## Reporting issues
Please open a GitHub Issue describing the problem, including the notebook cell
or script involved and, if applicable, the error message.

## Pull requests
If you'd like to propose a change:
1. Fork the repository
2. Create a feature branch (`git checkout -b fix/short-description`)
3. Make your change and verify the notebook still runs top-to-bottom
4. Open a pull request describing what changed and why
